🌐 ¿Qué es GitHub?

GitHub es una plataforma para guardar y compartir código. Usa un sistema llamado Git para llevar el control de versiones (es decir, los cambios en tus archivos).

También permite trabajar en equipo, colaborar y publicar tus proyectos de programación, como sitios web en HTML.

🚀 ¿Qué es GitHub Pages?

GitHub Pages es un servicio gratuito que ofrece GitHub para alojar sitios web estáticos (HTML, CSS, JS) directamente desde tu repositorio.

Ideal para portafolios, tareas, proyectos escolares o personales.

🛠️ Pasos para subir tu HTML a GitHub

  1. Crea una cuenta en github.com
  2. Haz clic en + New repository y nómbralo (por ejemplo, mi-sitio)
  3. En tu computadora, asegúrate de tener un archivo llamado index.html
  4. Haz clic en Upload files y sube tu index.html y otros archivos
  5. Haz clic en "Commit changes" para guardar

📢 Activar GitHub Pages

  1. En el repositorio, ve a Settings (Configuración)
  2. Busca la sección Pages o "Pages"
  3. En "Source", elige main y la carpeta / (root)
  4. Guarda. GitHub te dará un enlace como:
    https://tuusuario.github.io/mi-sitio

¡Y listo! Tu sitio HTML está publicado en internet.

📄 Archivos comunes que puedes subir

  • index.html – Página principal
  • style.css – Estilos
  • script.js – JavaScript
  • Imágenes, fuentes, sonidos, etc.

💡 Recuerda: GitHub Pages solo funciona con sitios estáticos (sin bases de datos ni PHP).
