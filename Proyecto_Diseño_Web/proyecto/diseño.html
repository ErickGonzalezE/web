Diseño y Desarrollo Web

Este módulo enseña herramientas y conceptos esenciales para la creación de contenido gráfico y páginas web atractivas y funcionales.

Herramientas utilizadas

  • GIMP: Editor de imágenes gratuito y potente, ideal para retoque y manipulación gráfica.
  • Canva: Plataforma online muy accesible para diseño gráfico, útil para crear presentaciones, infografías, etc.
  • Remove.bg: Herramienta en línea para eliminar el fondo de una imagen de forma automática.

Aplicaciones

  • Diseño de logotipos y banners
  • Maquetación de páginas web
  • Optimización de imágenes para la web
