🌐 ¿Qué es el Hosting?

El hosting es un servicio que proporciona espacio en servidores para almacenar los archivos de tu sitio web y que este sea accesible desde Internet. 🧠📡

Características:

  • Disponible las 24 horas del día. 🌙☀️
  • Espacio de almacenamiento y transferencia de datos. 💾
  • Soporte técnico, seguridad y respaldos. 🔒

¿Cómo funciona? Tu sitio se guarda en un servidor que se conecta constantemente a Internet. Cuando alguien escribe tu dirección web, accede a los archivos desde ese servidor. ⚙️🌍

Tipos de hosting:

  • 🔹 Compartido: varios sitios usan el mismo servidor (económico, pero limitado).
  • 🔹 VPS: servidor dividido en máquinas virtuales (más control).
  • 🔹 Dedicado: el servidor entero para un solo sitio (caro pero potente).
  • 🔹 En la nube: varios servidores trabajan juntos (escalable y seguro).

Ejemplos de servicios de hosting:

  • 🌐 Hostinger
  • 🌐 GoDaddy
  • 🌐 Bluehost
  • 🌐 InfinityFree (gratuito)
🐙 ¿Qué es GitHub?

GitHub es una plataforma en línea que usa el sistema de control de versiones Git. Sirve para guardar código, colaborar y trabajar en equipo en proyectos de software. 🚀👨‍💻

Características:

  • Control de versiones (historial de cambios). 🔄
  • Trabajo colaborativo con ramas (branches). 🤝
  • Almacenamiento y publicación gratuita de proyectos. 💻
  • Compatible con Git, VSCode, y otras herramientas. 🛠️

¿Cómo funciona? Subes tus archivos a un repositorio, que es como una carpeta en línea. Puedes trabajar solo o con otros, y todos los cambios quedan registrados. 📂🧠

¿Cómo subir una página web a GitHub? 🌐

  1. Crea una cuenta en GitHub.com.
  2. Crea un nuevo repositorio llamado nombreusuario.github.io.
  3. Sube tus archivos HTML, CSS y JS.
  4. Ve a "Settings" → "Pages" y selecciona la rama a publicar.
  5. Tu página estará en línea en: https://nombreusuario.github.io
