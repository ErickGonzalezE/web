Programación y Base de Datos (Visual Studio C#)

Este módulo introduce a los estudiantes en el desarrollo de aplicaciones utilizando el entorno de desarrollo Visual Studio y el lenguaje C#. Además, se abordan los fundamentos del diseño y la gestión de bases de datos relacionales como SQL Server.

Temas clave

  • Conceptos de programación orientada a objetos
  • Interfaz gráfica con Windows Forms
  • Conexión con bases de datos
  • Operaciones CRUD (Crear, Leer, Actualizar, Eliminar)
